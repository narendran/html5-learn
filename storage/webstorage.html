HTML5 WebStorage
This page will demonstrate web storage. Web Storage is a feature in HTML5 that allows you to store and load a key value pair on client side which will be persisted across application restarts. (Adheres to Same origin Policy).
How does it differ from Cookies? Ya I too had the same question. Cookies are also name-value pairs) are limited in size ie. 4 KB. But HTML5 storage allows a whopping 5MB for local storage and unlimited (limited only by system memory) for session storage. Here we have a demo of local storage.
